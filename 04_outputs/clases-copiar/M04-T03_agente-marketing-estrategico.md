# Tu agente: Marketing Estrategico PROEXCA

## Objetivo de aprendizaje
Al terminar esta clase, seras capaz de explicar que hace Marketing Estrategico PROEXCA, para que 3 tipos de documento esta diseñado, y como interactuar con el usando el patron GATE.

## Agenda
1. Que problema resuelve este agente (4 min)
2. Los 3 tipos de documento que produce (6 min)
3. Como hablarle: el patron GATE (5 min)
4. Metodologias que aplica por detras (5 min)

## Que problema resuelve

Imagina que manana tienes que preparar un dosier sectorial para una mision comercial al sector aeroespacial. Hoy, eso significa: buscar datos, estructurar el documento, redactar, ajustar tono institucional, revisar, maquetar. Minimo 3 dias de trabajo fragmentado.

Marketing Estrategico PROEXCA es tu Director de Marketing de internacionalizacion virtual. Le das el briefing y produce un documento completo — estructurado, con tono PROEXCA, adaptado al interlocutor — en 30-60 minutos.

No es un chatbot generico. Esta entrenado especificamente para los tres tipos de documento que un directivo de PROEXCA necesita producir con frecuencia.

## Los 3 tipos de documento

### Dosieres sectoriales (CU-01)
Informes de posicionamiento de Canarias en un sector para misiones comerciales. Estructura: resumen ejecutivo → contexto sectorial → oportunidades → datos clave → recomendaciones.

### Pitch decks (CU-02)
Presentaciones persuasivas adaptadas al perfil del interlocutor. El agente adapta la narrativa segun si hablas con un inversor, una empresa, o una institucion. Estructura: problema → oportunidad → solucion → evidencia → CTA.

### Kits de feria (CU-03)
Paquetes coordinados de materiales para eventos: nota de prensa (piramide invertida), ficha de stand (escaneable en 30 segundos), guion de presentacion. Las 3 piezas mantienen coherencia entre si.

> **Concepto clave:** Cada tipo de documento tiene su propia estructura y metodologia interna. No son "textos genericos" — son documentos profesionales con formato especifico.

## Como hablarle: el patron GATE

El agente no empieza a escribir hasta que tiene los inputs minimos. Esto se llama **patron GATE** — antes de producir, necesita que le des:

1. **Tipo** de documento (dosier / pitch / kit feria)
2. **Sector** o ambito (ej: "aeroespacial europeo")
3. **Destinatario** (ej: "inversores nordicos")
4. **Contexto** (ej: "mision comercial Hamburgo mayo 2026")
5. **Formato** deseado (si tienes preferencia)

Si no le das alguno, te lo preguntara. Esto evita que produzca documentos genericos — cada output esta adaptado a tu caso concreto.

## Metodologias que aplica

El agente no improvisa. Aplica metodologias reales por detras:
- **Piramide invertida** para notas de prensa — conclusion primero, detalles despues
- **Narrativa SCQA** para pitches — Situacion, Complicacion, Pregunta, Accion
- **Estructura de resumen ejecutivo** para dosieres — decisor primero, datos despues

Estas metodologias estan integradas en las instrucciones del agente. Tu no necesitas conocerlas — el agente las aplica automaticamente.

## Puntos clave
- Marketing Estrategico PROEXCA produce 3 tipos de documento: dosieres, pitches y kits de feria
- Usa el patron GATE: siempre dale tipo + sector + destinatario + contexto antes de pedirle que escriba
- Aplica metodologias profesionales por detras (piramide invertida, SCQA)
- No es generico — cada documento se adapta al interlocutor y al contexto especifico
