# Tu agente: Analista Estrategico PROEXCA

## Objetivo de aprendizaje
Al terminar esta clase, seras capaz de explicar que hace Analista Estrategico PROEXCA, para que 3 tipos de analisis esta diseñado, y como se complementa con Research & Insights.

## Agenda
1. El problema: informacion sin procesar, decisiones sin estructura (3 min)
2. Los 3 tipos de output que produce (5 min)
3. Como se complementan los dos agentes (4 min)
4. Patron GATE del analista (3 min)

## El problema

Los directivos de PROEXCA reciben informacion constantemente: informes de organismos, propuestas de colaboracion, estudios sectoriales, datos de mercado. El problema no es la falta de informacion — es la falta de procesamiento. Los documentos se acumulan, las decisiones se toman sin analisis formal, y las reuniones se preparan sobre la marcha.

Analista Estrategico PROEXCA resuelve esto: procesa informacion existente y la convierte en artefactos de decision claros. No investiga fuentes nuevas — analiza, sintetiza y estructura lo que ya tienes.

## Los 3 tipos de output

### Sintesis ejecutivas (CU-04)
Documentos complejos (informes, estudios, propuestas) → resumenes accionables de 1-2 paginas. Estructura: conclusion primero (BLUF), hallazgos clave, implicaciones para PROEXCA, recomendaciones. El directivo de Presidencia no lee 60 paginas — lee tu sintesis de 2.

### Notas de decision (CU-05)
Cuando hay que elegir entre opciones (mercados, sectores, propuestas, estrategias), el analista produce un documento comparativo. Estructura: contexto → opciones → tabla comparativa con criterios → pros y contras → riesgos → recomendacion. La decision queda documentada y comunicable.

### Briefings de contexto (CU-06)
Preparacion para eventos internacionales (misiones, ferias, reuniones con delegaciones). Estructura: contexto del evento → perfiles de interlocutores → temas clave → posiciones recomendadas → preguntas a hacer → proximos pasos. Llegas preparado, no improvisas.

> **Concepto clave:** Los 3 outputs comparten una regla: conclusion primero. El directivo lee las primeras 3 lineas y tiene el mensaje clave. Los detalles vienen despues, para quien los necesite.

## Como se complementan los dos agentes

| Dimension | Research & Insights | Analista Estrategico |
|-----------|-------------------|---------------------|
| Tipo de input | Pregunta de investigacion | Documentos existentes |
| Tipo de output | Inteligencia nueva | Analisis de info existente |
| Direccion | Mira hacia fuera | Mira hacia dentro |
| Analogia | Investigador | Consultor |
| Cuando usarlo | "No tengo datos de este mercado" | "Tengo datos pero no se que hacer con ellos" |

Si necesitas investigar un sector → Research & Insights.
Si necesitas sintetizar un informe → Analista Estrategico.
Si necesitas ambos → empiezas con Research (que trae los datos) y luego Analista (que los convierte en decision).

## Patron GATE del analista

Antes de analizar, Analista Estrategico necesita:
1. **Material** de entrada (documento, datos, opciones)
2. **Objetivo** del analisis (sintetizar, comparar, recomendar, contextualizar)
3. **Audiencia** del output (Presidencia, equipo, empresas)
4. **Formato** deseado (sintesis de N paginas, nota de decision, briefing)

Mas enfocado que Research — porque trabaja con material que ya existe, no con preguntas abiertas.

## Puntos clave
- Analista Estrategico PROEXCA produce 3 tipos de output: sintesis ejecutivas, notas de decision, briefings de contexto
- Conclusion primero, siempre — formato BLUF para directivos que no tienen tiempo
- Se complementa con Research & Insights: uno busca, el otro procesa
- Patron GATE: material + objetivo + audiencia + formato
