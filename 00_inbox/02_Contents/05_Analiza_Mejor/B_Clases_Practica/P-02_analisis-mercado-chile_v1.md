# Practica: analisis de mercado — Chile como destino de internacionalizacion

## Objetivo practico
Al terminar, tendras una ficha de mercado completa de Chile como destino de internacionalizacion para empresas canarias, lista para compartir con empresas interesadas.

## Supuesto (escenario realista)

Varias empresas canarias del sector tecnologico han preguntado a PROEXCA sobre oportunidades en Chile. El pais tiene un ecosistema tech en crecimiento, acuerdos comerciales con la UE, y una cultura de negocios relativamente afin.

Necesitas una ficha de mercado que responda las preguntas basicas: ¿que tamano tiene el mercado tech chileno? ¿como se entra? ¿que regulacion aplica? ¿quien esta ya alli? ¿que oportunidades concretas hay para empresas canarias?

La ficha debe ser practica y directa — las empresas la usaran como primera referencia para decidir si les interesa explorar Chile.

## Preparacion
Antes de empezar, asegurate de tener:
- El pais y sector de interes (Chile + tecnologia)
- El perfil de las empresas canarias que preguntan (tipo, tamaño, subsector)
- El objetivo (ficha de referencia para empresas)
- Acceso al agente **Research & Insights PROEXCA**

## Walkthrough paso a paso

### Paso 1: Briefing GATE al agente
Prompt sugerido:
> "Necesito una ficha de mercado de Chile como destino de internacionalizacion para empresas canarias del sector tecnologico. Objetivo: dar a las empresas una referencia practica para evaluar si les interesa Chile. Profundidad: media (ficha de 4-6 paginas, no estudio exhaustivo). Enfoque: oportunidades + canales de entrada + regulacion basica."

**Lo que debes ver:** El agente confirma el GATE y propone estructura: datos macro → sector tech chileno → regulacion y fiscalidad → canales de entrada → ecosistema local → oportunidades → riesgos → recomendaciones.

### Paso 2: Genera la ficha completa
Prompt sugerido:
> "Genera la ficha completa. Incluye una tabla comparativa al inicio con datos clave (PIB, poblacion, ranking Doing Business, acuerdo comercial con UE, idioma, diferencia horaria, conectividad desde Canarias). Marca [DATO: verificar] donde necesites cifras exactas."

**Lo que debes ver:** Ficha de 4-6 paginas con tabla comparativa rapida al inicio y secciones desarrolladas. Datos generales del mercado tech, empresas locales, programas de atraccion de inversion.

### Paso 3: Ajusta para el perfil canario
Prompt sugerido:
> "Ajusta la seccion de oportunidades para ser especifica sobre que tipo de empresa canaria encaja mejor: ¿software? ¿servicios IT? ¿ciberseguridad? Y añade las ventajas competitivas de Canarias para este mercado: zona horaria intermedia Europa-LATAM, zona franca, hub atlantico."

**Lo que debes ver:** Oportunidades desglosadas por subsector tech con argumentos de ventaja canaria.

### Paso 4: Añade la seccion practica "como empezar"
Prompt sugerido:
> "Añade una seccion final 'Primeros pasos para exportar a Chile' con 5 acciones concretas que una pyme canaria puede tomar: desde donde buscar socios hasta que programa de PROEXCA puede ayudarles."

**Lo que debes ver:** Seccion practica y accionable con pasos concretos y referencia a programas de apoyo.

## Ajuste fino
- La tabla comparativa del inicio es critica — un empresario decide si sigue leyendo en los primeros 30 segundos
- Verifica datos macro (PIB, poblacion, acuerdos) con fuentes oficiales
- Si la ficha se alarga, prioriza: oportunidades y "como empezar" son las secciones mas valiosas para una empresa

## Resultado esperado
Tu ficha de mercado final deberia tener:
- Tabla comparativa rapida (datos macro en un vistazo)
- Panorama del sector tech chileno
- Regulacion y fiscalidad basica para europeos
- Canales de entrada y ecosistema local
- Oportunidades por subsector tech canario
- Riesgos y consideraciones
- Seccion "Primeros pasos" con acciones concretas

## Errores comunes
- **Error:** Ficha demasiado generica ("Chile es un buen mercado")
  **Fix:** Especifica: ¿bueno para que tipo de empresa? ¿en que subsector? ¿con que ventaja?

- **Error:** No incluir datos comparativos
  **Fix:** Una tabla con 8-10 indicadores clave vale mas que 3 paginas de texto descriptivo

- **Error:** Olvidar la seccion practica
  **Fix:** Las empresas quieren saber "que hago manana" — siempre incluye primeros pasos concretos
