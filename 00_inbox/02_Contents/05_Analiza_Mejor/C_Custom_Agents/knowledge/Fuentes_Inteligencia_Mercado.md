# Fuentes de Inteligencia de Mercado: Guía Curada por Tipo de Research

## Propósito de este documento

Directorio curado de fuentes fiables para investigación de mercados e inteligencia comercial, organizado por tipo de necesidad. Actualizado para 2026.

## 1. DATOS MACRO Y ESTADÍSTICAS OFICIALES

### Organismos Estadísticos Nacionales

**Europa:**
- Eurostat: https://ec.europa.eu/eurostat
  - Datos oficiales UE: PIB, comercio, sectores económicos
  - Excelente para datos comparativos entre países UE

- Destatis (Alemania): https://www.destatis.de
- INE España: https://www.ine.es
- ONS (UK): https://www.ons.gov.uk
- INSEE (Francia): https://www.insee.fr

**América Latina:**
- CEPAL: https://www.cepal.org
- Bancos Centrales nacionales (datos macro actualizados)
- Institutos nacionales de estadística por país

**Global:**
- Banco Mundial - World Development Indicators: https://data.worldbank.org
- FMI - World Economic Outlook: https://www.imf.org/en/Publications/WEO
- OCDE: https://www.oecd.org/statistics/

**Nivel de confianza**: ALTO (fuentes primarias oficiales)

## 2. COMERCIO INTERNACIONAL

### Bases de datos de comercio

**Trade Map (ITC)**: https://www.trademap.org
- Estadísticas de comercio internacional por producto (código HS)
- Quién exporta qué a quién
- Aranceles y barreras comerciales
- GRATUITO (con registro)

**Nivel de confianza**: ALTO

**UN Comtrade**: https://comtrade.un.org
- Base de datos de comercio de la ONU
- Estadísticas detalladas de importación/exportación
- Históricos largos disponibles

**Nivel de confianza**: ALTO

**ICEX - DataComex**: https://www.datacomex.comercio.es
- Datos de exportación española por sector, país, comunidad autónoma
- Útil para benchmarking Canarias vs otras CCAA
- GRATUITO

**Nivel de confianza**: ALTO (datos oficiales España)

### Aranceles y regulación comercial

**Market Access Database (UE)**: https://madb.europa.eu
- Aranceles que enfrentan productos europeos en terceros países
- Acuerdos comerciales vigentes
- Barreras no arancelarias

**WTO - Tariff Analysis Online**: https://tao.wto.org
- Aranceles por país y producto

**Nivel de confianza**: ALTO

## 5. ASOCIACIONES SECTORIALES

### Metodología de búsqueda

Para encontrar asociación sectorial confiable:
1. Buscar: "[sector] + [país] + trade association"
2. Verificar: sitio web profesional, miembros identificables, informes con metodología
3. Nivel de confianza: MEDIO-ALTO (según transparencia)

## 11. LIMITACIONES Y PRECAUCIONES

### Fuentes a evitar o usar con extrema precaución

**Wikipedia:**
- ❌ NO citable en análisis profesional
- ✅ OK como punto de partida para explorar tema
- ✅ Revisar fuentes citadas por Wikipedia y citar esas

**Blogs personales sin credenciales:**
- ❌ Baja confianza
- ⚠ Solo usar si autor es experto reconocido

**Sitios sin fecha de publicación:**
- ❌ Imposible evaluar vigencia
- No usar o marcar [VERIFICAR]

**Informes de empresas vendiendo su producto:**
- ⚠ Verificar con fuentes independientes
- Sesgo comercial evidente

**Redes sociales (X/Twitter, LinkedIn):**
- ❌ NO citable
- ✅ OK para detectar tendencias emergentes
- ✅ Seguir a expertos reconocidos para estar al día

### Señales de alerta (Red Flags)

- Sitio con diseño amateur o lleno de anuncios
- No hay autor identificable
- No hay fecha de publicación
- No se citan fuentes de los datos
- Tono sensacionalista
- Cifras sin contexto o metodología

## 12. ESTRATEGIAS DE BÚSQUEDA EFECTIVA

### Para datos de mercado por sector y país

**Secuencia recomendada:**
1. Eurostat / Banco Mundial (datos macro)
2. Trade Map (datos comercio específico)
3. Asociación sectorial del país objetivo
4. Consultoras (McKinsey, Deloitte) para análisis
5. Medios especializados para tendencias

### Para validación de datos

**Regla de las 3 fuentes:**
- Buscar corroboración en al menos 3 fuentes independientes
- Si hay discrepancia, priorizar fuente oficial > consultora > medio > blog

### Para tendencias emergentes

1. Informes WEF, MIT Tech Review (macro)
2. Medios especializados sectoriales (tendencias específicas)
3. Ferias sectoriales (programas de conferencias)
4. Expertos en redes (detección temprana)

## 13. CHECKLIST: EVALUAR UNA FUENTE NUEVA

Cuando encuentres una fuente que no conoces:

```
┌─────────────────────────────────────────────────────────────────┐
│ CHECKLIST EVALUACIÓN DE FUENTE                                  │
├─────────────────────────────────────────────────────────────────┤
│ [ ] Autor/organización es identificable                         │
│ [ ] Sitio web es profesional y funcional                        │
│ [ ] Hay fecha de publicación clara                              │
│ [ ] Se citan fuentes de los datos                               │
│ [ ] Metodología es transparente (si aplica)                     │
│ [ ] El propósito es informar (no vender)                        │
│ [ ] Otros actores fiables citan esta fuente                     │
│ [ ] No hay red flags evidentes                                  │
│                                                                 │
│ Si 7-8 checks: Confianza ALTA                                   │
│ Si 5-6 checks: Confianza MEDIA (verificar con otra fuente)      │
│ Si < 5 checks: Confianza BAJA (no usar)                         │
└─────────────────────────────────────────────────────────────────┘
```
